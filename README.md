# Dubai Safari Tours

A professional and responsive **Single Page Application (SPA)** for booking desert safari tours in Dubai. Built with **React.js** and **React Router**, featuring dynamic pages, reusable components, and interactive UI notifications.

## Features

- Dynamic pages for tours, booking, and contact
- Responsive design using HTML5, CSS3, and Bootstrap icons
- Reusable React components for maintainable code
- Interactive notifications with React Toastify
- Smooth navigation using React Router
- Clean, scalable code structure

## Technologies Used

- React.js
- React Router
- JavaScript (ES6+)
- HTML5 & CSS3
- Bootstrap icons
- React Toastify
- Git for version control

## Installation & Running Locally

1. Clone the repository:

git clone <https://github.com/David-111409/Dubai-Safai-Tours-Project>

2- Navigate to the project folder:
cd dubai-safari-tours

3- Install dependencies:
npm install

4- Start the development server:
npm start
